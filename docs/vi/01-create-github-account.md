Hướng dẫn từng bước tạo tài khoản GitHub để bắt đầu hành trình lập trình và quản lý mã nguồn.

---

## 📋 Yêu cầu

| Yêu cầu | Chi tiết |
|----------|---------|
| 🌐 Trình duyệt web | Chrome, Edge, Firefox, hoặc Safari |
| 📧 Email | Địa chỉ email hợp lệ (chưa đăng ký GitHub) |
| 🔐 Mật khẩu | Tối thiểu 8 ký tự, bao gồm chữ hoa, chữ thường và số |
| 👤 Tên người dùng | Chỉ gồm chữ cái, số và dấu gạch ngang (không trùng) |

---

## 🚀 Các bước thực hiện

### Bước 1: Truy cập GitHub

Mở trình duyệt và điều hướng đến:

👉 **[https://github.com](https://github.com)**

Bạn sẽ thấy trang chủ GitHub với nút **"Sign up"** (Đăng ký) ở góc trên bên phải.

---

### Bước 2: Điền thông tin đăng ký

Trên trang đăng ký, nhập các thông tin sau:

| Trường | Hướng dẫn |
|--------|-----------|
| **Email** | Nhập địa chỉ email bạn muốn dùng. **Quan trọng:** Email này sẽ dùng để đăng nhập và nhận thông báo. |
| **Password** | Tạo mật khẩu mạnh (ít nhất 8 ký tự, có chữ hoa, chữ thường và số) |
| **Username** | Chọn tên người dùng duy nhất. Đây sẽ là địa chỉ GitHub của bạn (ví dụ: `github.com/ten-cua-ban`) |
| **Email preferences** | Chọn **Y** (có) nếu bạn muốn nhận email cập nhật sản phẩm, hoặc **N** (không) nếu bạn chỉ muốn email cần thiết |

> ⚠️ **Lưu ý:** Tên người dùng là vĩnh viễn — hãy chọn một cái tên chuyên nghiệp mà bạn muốn gắn bó lâu dài.

Sau khi điền xong, nhấn nút **Continue** (Tiếp tục).

---

### Bước 3: Xác minh (CAPTCHA)

GitHub sẽ yêu cầu xác minh bạn không phải robot:

1. Hoàn thành bài kiểm tra **CAPTCHA** (thường là chọn hình ảnh hoặc tích vào ô xác nhận)
2. Nhấn **Create account** (Tạo tài khoản)

---

### Bước 4: Xác nhận email

GitHub sẽ gửi mã xác nhận đến email của bạn:

1. **Mở hộp thư email** của bạn
2. **Tìm email** từ GitHub với tiêu đề "[GitHub] Please verify your email address"
3. **Nhập mã 8 chữ số** vào ô xác nhận trên trang GitHub
4. Nhấn **Verify** (Xác nhận)

> ⏳ Email có thể mất 1-2 phút để đến. Kiểm tra cả mục **Spam/Junk** nếu không thấy trong hộp thư chính.

---

### Bước 5: Chọn gói Free

Sau khi xác nhận email, bạn sẽ thấy trang chọn gói:

1. Chọn **"Free"** (gói miễn phí)
2. Nhấn **Continue**

Gói Free bao gồm:

| Tính năng | Gói Free |
|-----------|:--------:|
| Kho chứa không giới hạn (Public/Private) | ✅ |
| Cộng tác viên không giới hạn | ✅ |
| GitHub Actions (2000 phút/tháng) | ✅ |
| GitHub Pages (trang web tĩnh miễn phí) | ✅ |
| Codespaces (120 giờ/tháng) | ✅ |

---

### Bước 6: Hoàn tất Onboarding

GitHub sẽ hỏi một số câu hỏi khởi tạo:

1. **Số lượng thành viên nhóm:** Chọn "Just me" (Chỉ một mình) hoặc tùy theo nhu cầu
2. **Trình độ lập trình:** Chọn cấp độ phù hợp (Beginner / Intermediate / Professional)
3. **Quan tâm đến:** Chọn các chủ đề bạn muốn theo dõi
4. Nhấn **Complete setup** (Hoàn tất thiết lập)

> 💡 **Mẹo:** Bạn có thể bỏ qua các câu hỏi onboarding và trả lời sau bằng cách nhấn **"Skip personalization"**.

---

## ✅ Kết quả

Sau khi hoàn tất, bạn sẽ được đưa đến **Trang Dashboard** của GitHub:

- 🔗 Địa chỉ profile: `https://github.com/ten-cua-ban`
- 📊 Trang Dashboard hiển thị các repository, hoạt động gần đây
- ⚙️ Có thể truy cập Settings để tùy chỉnh thêm

---

## 🎉 Chúc mừng!

Bạn đã tạo thành công tài khoản GitHub! Giờ đây bạn có thể:

- 📦 Tạo repository để quản lý mã nguồn
- 🤝 Đóng góp vào dự án open source
- 🚀 Triển khai ứng dụng với GitHub Pages
- 👥 Cộng tác với đồng đội
